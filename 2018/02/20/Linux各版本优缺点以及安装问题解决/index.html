Linux各版本优缺点以及安装问题解决

发表于 2018-02-20

基于放假有时间，就捣鼓了一遍Linux各种版本。

基本全试过一遍后，debian、fedora、linuxmint、ubuntu、CentOS、openSUSE只要下的不是服务器版本的镜像，都是带有安装界面的引导，安装起来很方便，然后装CentOS时候手快点了最小安装没有图形界面，百度解决了一下，捣鼓最久的就是archlinux了。以下我说说个版本的优缺点，和我安装遇到的问题。(优缺点收集自百度)

Ubuntu

优点
  • 安装简单
  • Unity 3D图形界面，比较华丽（因人而异）
  • 固定的发布周期和支持期限
  • 丰富的文档
缺点
  • 硬件兼容性差（貌似所有Linux都有这个问题，部分硬件并不支持，后面的版本就不说这个缺点了）

Debian

优点
  • Free，最符合开源精神的发行版。默认不包含任何专有驱动
  • 不求新，但求稳。软件包通常选择比较稳定的版本而不是最新的版本
  • 系统的开发维护完全是由社区驱动的
  • APT软件包管理
  • 图形化安装方式对新手友好
缺点
  • 发行周期过长
  • 保守

Fedora

优点
  • YUM包管理
  • 新技术吸纳较快
  • 官网提供多种桌面环境镜像，可以满足不同桌面喜好的用户
  • 基本半年发布一个新版本
  • 图形化安装对新手友好
缺点
  • Fedora的优先目的往往偏向企业应用的特点，而不是桌面可用性
问题

起初使用的是默认分配1G内存的设置，结果安装过程中卡死了两次，随后改为了2G内存，便流畅安装。

openSUSE

优点
  • 华丽的KDE桌面
  • 安装镜像提供多个桌面环境
  • 包管理（YaST）倾向于图形化管理，图形界面用户友好程度要高一些
  • 官网提供常规版本和滚动发行版本
缺点
  • 中文社区匮乏
  • 对于新手来说或许安装会有些繁琐
问题

emmmm,安装完毕后进入系统，更改了系统语言还是有部分是英文的，当然，这只是我遇到的问题，不代表你会有。

CentOS

优点
  • 稳定可靠
  • 免费下载和使用
  • 配备了5年的免费安全更新
缺点
  • 缺乏最新的Linux技术
问题

(2)重启网络: service network restart

2.安装图形界面

(1)在终端中输入安装GNOME图形界面 yum groupinstall "GNOME Desktop" "Graphical Administration Tools"

(2)更新系统运行级别 ln -sf /lib/systemd/system/runlevel5.target /etc/systemd/system/default.target

(3)重启机器 reboot 就好了

LinuxMint

优点
  • 超强的“minty”工具集合
  • 有Ubuntu的特点
  • 操作界面更接近WindowsOS
缺点
  • 另类“community”社区版本并不总是包括最新功能，该项目不发出安全警告

ArchLinux

优点
  • 轻量级
  • pacman软件包管理工具
  • 灵活，整个系统完全由你说了算，当然你得有对应的能力
  • 完备的WIKI
  • WIKI中文化程度很高
缺点
  • 安装繁琐
  • 大部分都要网
问题
